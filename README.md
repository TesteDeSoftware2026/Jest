# Projeto de Teste de Software


## Requisitos

- Node.js 22+
- npm 10+

## Instalação

Clone o repositório:

```bash
# git bash
git clone https://github.com/TesteDeSoftware2026/Jest.git
```

Entre na pasta:

```bash
# git bash
cd ./Jest
code .
```

Instale as dependências:

```bash
# cmd
npm install
```

## Executar aplicação

```bash
# cmd
npm run dev
```

## Executar testes

```bash
# cmd
npm test
```
